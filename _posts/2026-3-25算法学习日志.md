## 1. Python 中的空值

在 Python 中没有 `null`，对应的是：

```python
x = None
if x is None:
    print("x is None")
```
类似其他语言：
* Java / C++ → null
* JavaScript → null
* Go → nil
## 2. Python 中深拷贝与浅拷贝
### 2.1 浅拷贝（Shallow Copy）
* 只复制最外层对象，内部嵌套对象仍然共享
* 常用方法：
```python
import copy
b = copy.copy(a)
#或
b = a[:]
b = list(a)
```
* 示例：
```python
a = [1, [2,3]]
b = a[:]
b[0] = 100
b[1][0] = 200
print(a)  # [1, [200,3]]
print(b)  # [100, [200,3]]
```
### 2.2 深拷贝（Deep Copy）
* 递归复制所有子对象，完全独立
* 常用方法：
```python
import copy
b = copy.deepcopy(a)
```
* 示例：
```python
a = [1, [2,3]]
b = copy.deepcopy(a)
b[1][0] = 200
print(a)  # [1, [2,3]]
print(b)  # [1, [200,3]]
```
## 3. 链表题的理解与刷题心得
### 3.1 Python 做链表题的特点
* Python 没有内置链表，需要自定义 ListNode
* 指针操作不直观，但安全
* 调试不如数组直观
* 写起来比 C++/Java 简洁
### 3.2 链表题的核心难点
1.指针操作：next 的重定向

2.边界条件：空链表、单节点、头节点变化

3.画图能力：帮助理解节点之间的连接
### 3.3 经典链表模板
反转链表（原地反转）
```python
prev = None
cur = head
while cur:
    nxt = cur.next
    cur.next = prev
    prev = cur
    cur = nxt
return prev
```
快慢指针找中点/环
```python
slow = fast = head
while fast and fast.next:
    slow = slow.next
    fast = fast.next.next
```
虚拟头节点（dummy）
```python
dummy = ListNode(0)
dummy.next = head
```
### 3.4 常见坑
* 赋值不等于拷贝

b = a  # 只是引用，修改 b 会影响 a

* 滥用 deepcopy 会导致性能和逻辑问题
### 3.5 面试链表题训练建议
1.反转链表

2.合并两个有序链表

3.删除指定节点

4.环形链表判断（快慢指针）

5.K 个一组翻转链表（进阶）

练习方法：

画图理解每一步操作

多次复写，加深对 prev / cur / nxt 的理解

理解比死记模板更重要，但重复练习可以把理解变成反射

## 4. 链表题学习心得
* “不会” ≠ “不理解”，可能只是模板没有形成条件反射
* 模板是理解的高度压缩，不是死记
* 练习重点：把理解 → 模板 → 自动化反射
* 目标不是记住每道题，而是掌握底层操作模型

## 总结：

Python 中空值用 None，深浅拷贝掌握原则

链表题关键在指针操作、边界条件和画图理解

掌握反转链表、快慢指针、虚拟头节点等模板

练习多次，把理解转化为“自动反应”

面试链表题不怕忘记模板，理解底层原理最重要
