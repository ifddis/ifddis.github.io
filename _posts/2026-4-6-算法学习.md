# 📅 算法学习日志（矩阵 + 二分查找 + 递归理解）

## 🧩 一、螺旋矩阵问题

### 题目

按顺时针顺序输出矩阵元素。

---

### 💡 我的思路（旋转 + 递归）

* 每次取第一行
* 剩余矩阵逆时针旋转
* 递归处理

---

### ✅ 核心代码

```python
def spiralOrder(matrix):
    res = []
    
    while matrix:
        res += matrix[0]
        matrix = [list(row) for row in zip(*matrix[1:])][::-1]
    
    return res
```

---

### 🧠 核心思想

> 把“多方向遍历”转化为“始终取第一行”

---

### ⚠️ 问题

* 时间复杂度：O(n³)
* 空间复杂度：O(n²)
* 每一层都在复制矩阵（性能差）

---

### ✅ 面试最优解（边界法）

* 使用 `top, bottom, left, right`
* 一圈一圈遍历

👉 时间复杂度：O(mn)，空间 O(1)

---

## 🧩 二、矩阵旋转（原地）

### 题目

将 n×n 矩阵顺时针旋转 90°（原地）

---

### 💡 我的解法

* 使用新矩阵（不符合“原地”要求）

---

### ✅ 标准解法

#### 两步走：

1. 转置
2. 每一行反转

```python
for i in range(n):
    for j in range(i+1, n):
        matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]

for row in matrix:
    row.reverse()
```

---

### 🧠 本质

> 坐标变换：(i,j) → (j,n-1-i)

## 🧩 三、矩阵搜索（有序矩阵）

### 题目特点

* 行递增
* 列递增

---

### 💡 我的思路

* 每一行二分查找

👉 时间复杂度：O(m log n)

---

### 🚀 更优解（面试推荐）

从右上角开始：

* 大了 → 左
* 小了 → 下

```python
i, j = 0, n-1
while i < m and j >= 0:
    ...
```

👉 时间复杂度：O(m + n)

## 🧩 四、二分查找

### ✅ 标准模板（迭代）

```python
def binary_search(nums, target):
    left, right = 0, len(nums)-1
    
    while left <= right:
        mid = (left + right)//2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

---

### ⚠️ 常见错误

* `while left < right` ❌
* `left = mid` ❌（死循环）
* 忘记 `+1 / -1`

---

## 🧩 五、递归二分查找问题

### ❌ 我的问题

```python
biosearch(lst[mid+1:], target)
```

👉 没有 `return` → 返回 None

---

### ❗ 修复

```python
return biosearch(...)
```

---

### ⚠️ 关键难点：索引偏移

右半边：

```python
lst[mid+1:]
```

👉 返回索引需要：

```python
mid + 1 + res
```

---

### ❓ 为什么左边不用？

```python
lst[:mid]
```

👉 索引从 0 开始 → 与原数组一致

---

### 🧠 本质理解

> 切片改变了索引起点

---

### 🚀 更优写法（推荐）

不使用切片，直接传索引：

```python
def binary_search(nums, target, left, right):
    if left > right:
        return -1
    mid = (left + right)//2
    if nums[mid] == target:
        return mid
    elif nums[mid] < target:
        return binary_search(nums, target, mid+1, right)
    else:
        return binary_search(nums, target, left, mid-1)
```

## 🧩 六、zip(*matrix) 理解

### ❌ 我的误区

```python
list(i for i in zip(*matrix))
```

👉 以为 tuple 会变 list ❌

---

### ✅ 正确写法

```python
[list(i) for i in zip(*matrix)]
```

---

### 🧠 本质

> list() 只作用在最外层，不会递归转换
